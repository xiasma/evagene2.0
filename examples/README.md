# Example Pedigrees

XEG files for demonstrating the BayesMendel risk analysis models.
Import via the `.xeg` button in the toolbar.

| File | Model | Scenario |
|------|-------|----------|
| `brca_family.xeg` | BRCAPRO | Hereditary breast/ovarian cancer. Proband (Sarah, 42) has mother and aunt with breast cancer, grandmother with ovarian cancer. |
| `lynch_family.xeg` | MMRpro | Lynch syndrome. Proband (Rachel, 38) has father and uncle with colon cancer, grandmother with endometrial cancer. |
| `pancreatic_family.xeg` | PancPRO | Familial pancreatic cancer. Proband (Tom, 50) has mother and uncle with pancreatic cancer. |

Each file is a three-generation pedigree with birth dates, death dates,
and cancer diagnoses at realistic ages. Diseases are defined within the
XEG and will be matched to the catalog on import.
